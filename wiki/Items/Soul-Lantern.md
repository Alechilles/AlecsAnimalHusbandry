# Soul Lantern

Parent: [Items Index](/mod/alecs-animal-husbandry/items-index) | [Home](/mod/alecs-animal-husbandry/animal-husbandry-wiki)

The **Soul Lantern** lets you capture and re-place your tamed livestock and Beast companions.
It is useful for moving companions between pens, relocating them to a new base, or packing them up before travel.
Unlike a simple spawn item, it stores the animal's saved data so the companion is not reset when placed again.

## Crafting
- Crafting station: **Farmer's Workbench (Tier 2)**
- Recipe:
  - 6x **Any Wood**
  - 2x **Void Essence**
- Craft time: **3 seconds**
- Max stack: **1**

## How To Use
1. Hold an **empty Soul Lantern**.
2. Interact with a valid **tamed** livestock animal or Beast you own to capture it.
3. Move to the new location where you want to place that animal.
4. Use the filled lantern to release it back into the world.

## Default Rules
- Only works on supported **tamed livestock** and **tamed Beasts**.
- You can only capture companions that are already tamed.
- Ownership rules apply, so this is intended for **your own animals**.
- The lantern stores **one animal at a time**.
- Capture and release range is **short**, so stand fairly close when using it.

## Preserved Data
The **Soul Lantern** preserves the stored animal's metadata instead of turning it into a fresh default copy.

- **Names** are kept.
- **Skins and appearance variants** are kept.
- **Traits** are kept.
- **Happiness state** is kept.
- **Needs state** is kept.
- And more...

## Ownership Behavior
When an animal is captured, its ownership is cleared while it is stored inside the lantern.
When that animal is released again, ownership is assigned to the player who places it.

This makes the **Soul Lantern** useful for trading companions between players while still preserving the animal's saved data, appearance, and progression.

## Good Uses
- Move a companion to a new pasture or enclosure.
- Reposition a Beast without walking it across dangerous terrain.
- Pick up companions before rebuilding fences, barns, or habitat layouts.
- Transport valuable breeding stock without leading them one by one.
