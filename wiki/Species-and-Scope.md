# Species and Scope

## Livestock Focus (Primary)
The most mature progression loop is livestock husbandry.

Current livestock line set includes:
- Bison
- Boar / Pig / Warthog lines
- Bunny / Rabbit
- Camel
- Chicken / Desert Chicken
- Cow
- Goat
- Horse
- Mouflon / Sheep / Ram lines
- Skrill
- Turkey

## Beast Coverage
Beast and beast companions are fully integrated into command and trait systems.
They also have dedicated per-role breeding pacing.

Examples in current Beast set:
- Bears
- Crocodile
- Emberwulf / Fen Stalker
- Fox / Hyena
- Raptor / Rex
- Snakes / Spiders / Scorpion
- Tiger Sabertooth
- Toad Rhino variants
- Wolves
- Yeti
